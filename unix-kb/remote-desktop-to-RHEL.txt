REMOTE DESKTOP TO RHEL/OEL MACHINE FROM YOUR PC

DOWNLOAD VNC VIEWER
https://www.realvnc.com/download/viewer/
NOTE: If you have admin-access to your PC, download exe and install it.  
      Otherwise, download zip version and run the program in it.
      
CONFIGURE REMOTE DESKTOP ON RHEL MACHINE
System --> Preferences --> Remote Desktop
In 'Sharing' - 
check - allow other users to view your desktop
check - allow other users to control your desktop
