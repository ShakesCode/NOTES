======
BASICS
======

Playbook file is named like filename.yml

To execute it:
$ ansible-playbook -s  filename.yml -u vagrant

============
EXAMPLE 1
============
---
- hosts: 192.168.4.101
  vars:
    http_port: 80
    max_clients: 200

  become: true
  #remote_user: root
  tasks:
    - name: Just cats the passwd file to stdout
      shell: cat /etc/passwd
