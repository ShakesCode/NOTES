=============================
PUSH CHANGES TO MASTER
=============================

SSH KEYS
Ensure .ssh keys point to the right git repository (public or company as needed)

CLONE (IF NOT DONE ALREADY)
git clone <repository URL>

Example:
git clone git@github.com:gowrishdba/cblamp1.git

It clones to directory named same as the repository

DO CHANGES
Edit files as needed in local cloned folder

PUSH
git add .
git commit -m "message"
git push origin master
