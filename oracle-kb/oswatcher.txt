OSWATCHER NON-ROOT VERSION

INSTALL
tar xvfp oswbb733.tar

RUN
./startOSWbb.sh 120 48 gzip /oracle/product/oswatcher/archive

120 is the interval to run in seconds
48 is days of retention of output
gzip is to direct it to zip the output
The 'archive' folder is where outputs are kept
