=============================
INSTALL GROOVY ON MAC
=============================

Download:
http://groovy-lang.org/install.html#download-groovy
--> https://bintray.com/artifact/download/groovy/maven/apache-groovy-binary-2.4.9.zip

Unzip the file:
Unip the zip file into /Users/username/opt (or other location)
--> unzips into subfolder groovy-2.4.9

Set environment:
export GROOVY_HOME=/Users/username/opt/groovy-2.4.9
export PATH=$PATH:$GROOVY_HOME/bin
