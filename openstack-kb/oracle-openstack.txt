Control and Compute Nodes
-------------------------
http://www.oracle.com/technetwork/server-storage/openstack/linux/downloads/index.html
An OpenStack deployment consists of compute and control nodes
      Control nodes are installed on Oracle Linux 6.5 or higher 
      For compute nodes user can use Oracle VM server 3.3.1 or KVM installed on Oracle Linux 6.5 or higher
